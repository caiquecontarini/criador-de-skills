# 🏭 Criador de Skills Automático

> **Uma skill que cria outras skills.** Automação de automação — o próximo nível.
> Compatível com Claude Code e agentes de IA baseados em MCP.

---

## 🚀 O Que É Isso?

Esta skill transforma o seu agente de IA em uma **fábrica de habilidades**. Em vez de criar skills manualmente, você descreve o que precisa e o agente escreve o `SKILL.md` completo, otimizado e pronto para uso.

---

## ✨ Funcionalidades

- **🎯 Geração Guiada:** Descreva a skill em linguagem natural — o agente estrutura tudo.
- **✅ Validação Automática:** Verifica se a skill gerada segue os padrões de qualidade.
- **📦 Pronto para Deploy:** Output direto no formato `SKILL.md` compatível com qualquer agente MCP.
- **🔄 Iteração Rápida:** Refine a skill com feedbacks simples em linguagem natural.

---

## 🛠️ Como Usar

```
1. Instale esta skill no diretório skills/ do seu agente
2. Peça ao agente: "Crie uma skill para [descreva a função]"
3. O agente vai gerar o SKILL.md completo
4. Revise, ajuste se necessário e instale a nova skill
```
